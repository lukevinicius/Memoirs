
# Memoirs

Application to save your daily memories, produced in React Native with Typescript.


## Run Locally

Clone the project

```bash
  git clone https://github.com/lukevinicius/Memoirs
```

Go to the project directory

```bash
  cd Memoirs
```

Install dependencies

```bash
  Yarn add
```

Start the server

```bash
  yarn android
  yarn start
```
